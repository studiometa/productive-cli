# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-16

### Added
- Initial release of @studiometa/productive-cli
- Configuration management with XDG compliance
- Projects management (list, get)
- Time entries management (list, get, add, update, delete)
- Tasks management (list, get)
- People management (list, get)
- Services management (list)
- Budgets management (list)
- Multiple output formats: JSON, CSV, Table, Human-readable
- AI agent optimization with structured JSON output
- Native Node.js implementation (zero runtime dependencies)
- Environment variable support
- Built-in help and version commands
- Pagination support
- Filtering and sorting capabilities
- TypeScript support
- Comprehensive test suite with Vitest
- GitHub Actions CI/CD workflows
- oxlint and oxfmt integration
